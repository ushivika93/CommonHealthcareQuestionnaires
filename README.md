# CommonHealthcareQuestionnaires

SAS programs and original articles to automatically score commonly used healthcare psychology questionnaires.

## Available Questionnaires

- **ACE** - Adverse Childhood Experiences (10 items)


## How to Use

1. Download the .sas file for your questionnaire
2. Make sure your dataset variable names match the script
3. Please be on the lookout for annotations in the program for further instructions.
4. Run the program in SAS
5. Your scored data will be ready
